## 1.1 go语言实现一个Fibonacci数列
```
func TestFib(t *testing.T) {
	a:=1
	b:=1
	t.Log(a)
	for i:=0;i<5 ;i++  {
		t.Log("",b)
		temp:=a
		a=b
		b=temp+a
	}
}
```
## 1.2 go语言多变量赋值 ，一个赋值语句中实现对多个变量赋值
```
func TestExchange(t *testing.T)  {
	a:=1
	b:=2
	a,b=b,a
	t.Log(a,b)
}
```

## 2.2 第一个比特位为1 其他位为0 表示Open 依此类推...
```
const (
	Open =1 << iota
	Close
	Pending
)
```
